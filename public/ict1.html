Generation of Computers

The computer was born in the attempt to make an adding machine. To add numbers, a device called the abacus was invented.

In 1642, Blaise Pascal invented a machine called the adding machine, which was the world's first-ever mathematical machine.

In 1674, Gottfried Wilhelm von Leibniz improved the adding machine, making it capable of performing multiplication and division.

A French scientist named Joseph Jacquard invented a mechanical loom using the punch card system.

Charles Babbage started to make his Analytical Engine using the punch card system concept. This machine was based on the concept of input, process, storage, and output. Since this concept helped develop the computer, he is known as the father of the computer.

Madam Ada Augusta Lovelace is considered the world's first programmer since she attempted to write programs for the Analytical Engine.

1st Generation

Major Hardware

  • Vacuum tubes
  • Punch cards were used for input, output, processing, and storage of data

Characteristics

  • High heat generation
  • Slow processing
  • Large in size
  • Not portable
  • Consumes a lot of energy

Software Used

  • Machine language
  • Assembly language
  • Stored program concept

Systems Invented

  • ENIAC
  • EDVAC
  • EDSAC
  • UNIVAC
  • IBM 700

2rd Generation

Major Hardware

  • Integrated circuit (IC)
  • Keyboard and mouse were used for input
  • High-capacity disks for secondary storage

Characteristics

  • Smaller in size
  • Less heat generation
  • Low power consumption
  • Comparatively faster than 2nd generation
  • Expensive

Software Used

  • Birth of the operating system
  • Well-developed programming languages
  • High-level computer language for coding

Systems Invented

  • IBM 360/370
  • PDP-11
  • PDP-8
  • CDC 6600

3th Generation

Major Hardware

  • Large-scale integrated circuit (LSIC)
  • Very large-scale integrated circuit (VLSIC)
  • Microprocessor
  • High-capacity hard disks
  • Floppy disks
  • Optical disks

Characteristics

  • Very small in size
  • Portable
  • Upgradeable
  • Faster computer networks

Software Used

  • OS with GUI
  • UNIX OS

Systems Invented

  • IBM PC
  • Apple II
  • Personal computer (PC)
  • Palm laptops

4th Generation

Major Hardware

  • Ultra large-scale integration (ULSI)
  • Very large capacity hard disks and optical disks
  • Internet

Characteristics

  • Smaller in size
  • Portable
  • High reliability
  • High efficiency
  • Less expensive
  • Easy operation

Software Used

  • OS with GUI
  • Internet and multimedia applications
  • Voice recognition based on AI
  • Character recognition
  • Handwriting recognition systems

Systems Invented

  • IBM notebooks
  • Pentium PC
  • Sun Workstation 4

Chromatography Explained

You can use paper chromatography to identify the particular dyes in a mixture.

We can see that all the compounds in the mixture corresponding to the dyes have traveled. Using this, we can say that they are the same compounds.

However, if you look at dye 2, the compound does not travel the same distance as the compounds in the mixture, so we can say that the compound is different.

Each time the experiment is repeated, the dyes will move differently.

We cannot work with the distance traveled alone; we have to come up with a ratio.

This ratio is known as the retention factor.

R₁ = retention factor

R₁ = distance moved by a spot (from the pencil line) / distance traveled by the solvent.

Since the spots have the same R₁ value as dyes 1, 2, and 3, we can say the three components are dyes.

The R₁ value must be between 0 and 1.

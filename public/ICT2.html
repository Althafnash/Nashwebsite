Operating System

A computer consists of software, hardware, and firmware.

Any physical component of the computer with a defined shape is known as hardware.

The booting instructions stored in the ROM (Read-Only Memory) are known as firmware.

How the Initial Operations of a Computer are Performed

  • When the user powers up the CPU, it activates the BIOS (Basic Input Output System).
  • The first program activated is POST (Power-On Self Test) using the CMOS (Complementary Metal-Oxide-Semiconductor).
  • After that, it reads the MBR (Master Boot Record) on the boot drive in accordance with the bootstrap loader provided by the computer manufacturer.
  • Then the computer loads the operating system into the RAM.
  • Once this is performed, the operating system takes control and displays a user interface to the user.

The process of loading the operating system into the RAM is known as booting.

Software is a set of instructions given to the computer to perform a task.

System software can be divided into three parts:

  • Operating system
  • Utility software
  • Language translators

The operating system provides the user with the ability to utilize the functions of a computer by managing both hardware and software.

Utility software is used to manage and analyze the software on the computer. Unlike application software, utility software differs in its complexity and operational activities.

A computer program (software) is made up of a set of instructions written in a high-level language, which is closer to human language.

Some examples of operating systems are:

  • Microsoft Windows
  • Apple macOS
  • Ubuntu Linux
  • Android OS

Operating systems can be classified according to their functionality:

  • Single-user operating systems
  • Multi-user operating systems
  • Multi-tasking operating systems
  • Real-time operating systems

The operating system that provides services to one person at a time is known as a single-user operating system.

The operating system that allows multiple users to use the system simultaneously is known as a multi-user operating system.

An operating system that allows running multiple processes at the same time is called a multi-tasking operating system.

Real-time operating systems are systems that provide output in real time without visible delay.

Services of an operating system include:

  • Managing the hardware of a computer
  • Providing an interface for user interaction

The hardware of a computer is managed using the following processes:

  • Process management
  • Memory management
  • Device management
  • File management
  • Security management
  • Network management

A running computer program or part of a program is called a process.

All activities in a computer run as either a single process or as multiple processes.

Resource management activities include the allocation of CPU time, memory, and I/O devices to each process.

Ordering of processes (according to a sequence) is also performed under this management.

Memory plays a crucial role in the functioning of a computer.

The input data is stored in the RAM before being transferred to the CPU, and processed data is also stored in the RAM before being sent to the output devices.

Systematic management of memory is vital for the proper functioning of a computer.

The operating system utilizes both primary memory (RAM) and secondary memory efficiently to manage memory for various processes.
